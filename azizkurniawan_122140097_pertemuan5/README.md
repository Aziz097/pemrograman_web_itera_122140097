## Cara Menjalankan

1. (opsional) Buat virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ````

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Jalankan:

   ```bash
   python main.py
   ```
